## 📄 dl-paper: Deep Learning for Everyone

This paper traces the arc of deep learning from its earliest ideas in the 1950s to the LLMs making headlines today, arguing that deep learning is very much alive — ChatGPT, Claude, and DeepSeek are all deep learning.
